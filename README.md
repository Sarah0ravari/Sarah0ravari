### [HardScope Creator Campaign Analytics](https://github.com/Sarah0ravari/hardscope-creator-campaign-analytics)
A creator campaign analytics application built with real YouTube performance data.

- Presents a React dashboard for brand partnerships decision-making

### [Mood-Queue](https://github.com/yourusername/mood-queue)
A lightweight internal tool for logging and visualizing the emotional “vibe” of support tickets throughout the day.
- Built with Streamlit, Google Sheets, and Plotly
- Lets support agents log moods in real time using emoji-based inputs
- Stores entries with timestamps and optional notes
- Visualizes daily sentiment trends to help teams spot support friction early
